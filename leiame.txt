Instruções:

  Para compilar, digite: make all
  Para executar o programa juntamente com o Valgrind, digite: make run
  Para ver o relatorio do Valgrind, abra: valgrind_out.txt

  Obs.: 
  1. make all, além de compilar ira executar as checagens do
  cppcheck, cpplint e mostrar os resultados dos testes pelo catch.
  2. É necessario ter instalado o catch, o cppcheck e o cpplint no 
  computador.
  3. Use o compilador g++ 7.3.0; 
  4. Use o SO Linux;
